Este proyecto corresponde a una aplicación web desarrollada en HTML, CSS y JavaScript, cuyo objetivo es permitir el registro de jugadores para un torneo de básquet.

El sistema solicita al usuario ingresar:

Nombre

Edad

Altura (en centímetros)

Posición preferida

Luego valida los datos ingresados y determina si el jugador cumple con los requisitos mínimos para inscribirse. En caso de cumplirlos, el sistema lo clasifica automáticamente en una categoría.

Contexto Académico

Proyecto desarrollado para la asignatura de JavaScript, aplicando:

Manipulación del DOM

Validaciones

Eventos

Estructura básica de aplicación web
